# 📖 Project Overview

The **OLA Ride Booking Analysis** project demonstrates an end-to-end Business Intelligence workflow using **Microsoft Excel**, **SQL**, and **Power BI**. The project focuses on transforming raw ride booking data into meaningful insights through data preparation, SQL analysis, KPI tracking, and interactive dashboards.

Using approximately **40,000 ride booking records**, the project analyzes booking trends, vehicle performance, cancellations, revenue, and customer satisfaction to support data-driven business decisions.

---

# 🎯 Business Problem

Ride-hailing companies generate thousands of booking records every day. Analyzing this data is essential for understanding customer behavior, improving operational efficiency, optimizing fleet allocation, and monitoring business performance.

This project provides an interactive dashboard that enables stakeholders to track key metrics and make informed decisions based on real-time insights.

---

# 🎯 Project Objectives

- Analyze ride booking performance and trends.
- Monitor key business performance indicators (KPIs).
- Evaluate vehicle-wise booking performance.
- Analyze ride cancellation patterns.
- Study customer and driver ratings.
- Build an interactive dashboard for business reporting.
